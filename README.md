## 畫面中，最上面的數字是亂數，也是答案，因為方便測試，所以就顯示出來。  

## 功能：
- 答案為亂數
- 只能猜6次，6次之後要按" 再玩一次 "才能再玩
- 答對之後即停止，要按" 再玩一次 "才能再玩
- 輸入50之後的數字，會有提示文字
- 不輸入數字，會有提示文字
- 每次輸入前都會清空輸入框
- " 再玩一次 " 是 func
